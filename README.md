# מאזן חכם - BVA

אפליקציית תקציב אישי עם מטריצת תחזית חודשית.

## הרצה מקומית

```bash
# התקנת תלויות
npm install

# הרצת client + server יחד
npm run dev
```

האפליקציה תיפתח ב:
- **Client**: http://localhost:3000/
- **Server API**: http://localhost:3001/

## מבנה הפרויקט

```
BVA/
├── client/          # React + Vite + TypeScript
│   └── src/
│       └── components/

├── server/          # Express + TypeScript

│       └── routes/  # API endpoints
└── shared/          # Types משותפים
```

## תכונות עיקריות

### 1. מטריצת תקציב (Budget Matrix)
- **עמודות** = חודשים (12/25, 01/26, 02/26...)
- **שורות** = קטגוריות (מעונות, רכב, משרד...)
- **תאים** = סכום בפועל לכל קטגוריה בכל חודש
- **צבעי רקע** = לפי קבוצת הוצאות (כל קבוצה בצבע משלה)

### 2. מצבי תצוגה
- **תחזית** (ברירת מחדל) - מציג את התקציב לכל החודשים
- **בפועל** - מציג את הביצוע בפועל כשיש, אחרת את התקציב

### 3. עריכה ישירה (Inline Editing)
במצב "בפועל":
- **קליק על תא** → נפתח שדה עריכה
- **Enter** → שומר
- **Escape** → מבטל
- **לחוץ מחוץ לשדה** → שומר אוטומטית

### 4. הוספת תנועה חדשה
- כפתור "+ הזן בפועל" או קליק על תא במצב תחזית
- בחירת קבוצה → קטגוריה → סכום + חודש
- אפשרות ליצור קבוצה/קטגוריה חדשה

### 5. חיווי חריגה
- **▲ אדום** - כאשר הבפועל חורג מהתקציב
- **צבע תא** - הפרש בין תקציב לבפועל
